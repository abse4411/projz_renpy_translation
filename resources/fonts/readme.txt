You can put fonts here, then config your font paths in config.yaml.
We have configed three default fonts, and you can download them from the following links:

DejaVuSans.ttf: https://github.com/renpy/renpy/blob/master/renpy/common/DejaVuSans.ttf
SourceHanSansLite.ttf: https://github.com/renpy/renpy/blob/master/sdk-fonts/SourceHanSansLite.ttf
Roboto-Light.ttf: https://github.com/renpy/renpy/raw/master/launcher/game/fonts/Roboto-Light.ttf

PS: You SHOULD exercise caution in copyright issues with the use of fonts.
